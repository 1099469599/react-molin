# react-molin
[react-molin](https://github.com/Molin123/react-molin)是一个基于[webpack](https://github.com/webpack/webpack)和[antd-mobile](https://github.com/ant-design/ant-design-mobile)的移动端开发框架


# 项目依赖
* webpack^2.3.2
* antd-mobile^1.0.7
* react^15.4.2
* react-router-dom^4.0.0
